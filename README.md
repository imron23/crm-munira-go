# Munira CRM - Umrah SaaS (Go + Next.js Migration)

Sistem CRM khusus Travel Umrah Munira World yang telah dimigrasi dari Node.js/Express ke arsitektur modular menggunakan **Golang** dan **Next.js (App Router)**.

## Tech Stack
- **Frontend**: [Next.js 14](https://nextjs.org/) (TypeScript, Tailwind CSS, Chart.js)
- **Backend**: [Go (Golang)](https://go.dev/) (Gin Gonic, PostgreSQL, Domain Driven Design)
- **Database**: [PostgreSQL](https://www.postgresql.org/)
- **UI/UX**: Custom Glassmorphism Dark Mode

## Struktur Project
- `/frontend`: Aplikasi Next.js (Port Default: 5600)
- `/backend`: API Go (Port Default: 8080)
- `/migration-docs`: Dokumentasi skema DB dan rencana migrasi

## Cara Menjalankan di Lokal
1. **Backend**: 
   - `cd backend`
   - `go run cmd/api/main.go`
2. **Frontend**: 
   - `cd frontend`
   - `npm run dev`
